# Instructions

Compute the best possible poker hand that can be made from a provided hand of 5 cards.

Standard poker hand rankings determine the strength of a hand.

Cards are represented as two-character strings: the first character for rank (A, K, Q, J, T, 9-2) and the second for suit (c, d, h, s).

Output hand should be sorted by rank high-to-low and then by suit alphabetically.
If the input is not valid, print "Input is not valid." For example, hand does not include exactly 5 unique valid cards

Poker Hand Rankings (descending order): 
- Straight flush
- Four of a kind
- Full house
- Flush
- Straight
- Three of a kind
- Two pair
- One pair
- High card

Example input:
``` json
["Ac","Kc","Qc","Jc","Td"]
```

Example output:
``` text
Straight  ["Ac","Kc","Qc","Jc","Td"]
```
